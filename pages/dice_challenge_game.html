Short Description

A web-based mini project developed using JavaScript, HTML, and CSS. The idea was to create an interactive dice game where two dice roll and compete against each other to determine a winner.
The main goal of this project was to learn how to connect JavaScript functionality with HTML structure and CSS styling, as well as practicing layout and design techniques in CSS.
