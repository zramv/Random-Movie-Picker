Short Description

A basic blog project built without using a database, focusing on server-side rendering with EJS (Embedded JavaScript).
The main objective of this project was to learn how to use EJS templates to dynamically generate HTML content and avoid repeating HTML code across pages.
This involved creating reusable components such as headers, footers, and post templates, and rendering them efficiently through routes in Node.js.
The project helped me understand the fundamentals of templating engines, route handling, and how to separate logic from layout in web applications.
It was a key step toward building more structured and maintainable server-rendered apps.
