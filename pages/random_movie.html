Short Description

A web project built using Node.js and EJS (Embedded JavaScript Templates), where users can click a button to display a random movie along with its rating.
The purpose of this project was to learn how to work with external APIs and dynamically render JSON data into EJS templates instead of writing repetitive HTML code.
Through this project, I practiced making asynchronous requests, handling JSON data, and integrating the results into server-rendered pages.
It also helped me improve my skills in route handling and passing data from the server to the templates to create a more interactive user experience.
